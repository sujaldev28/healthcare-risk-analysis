# 🏥 Healthcare Risk Analysis

## 📌 Overview
This project focuses on analyzing healthcare datasets to identify **key risk factors** for early diagnosis of diseases such as diabetes and heart disease.

It leverages **data analysis, statistical methods, and machine learning** to uncover patterns and support data-driven healthcare decisions.

---

## 🎯 Objectives
- Identify significant **risk factors** affecting disease outcomes  
- Perform **exploratory data analysis (EDA)**  
- Apply **statistical techniques** for validation  
- Build **predictive models** for early diagnosis  
- Generate **actionable insights** for healthcare improvement  

---

## 📊 Dataset
- Source: Kaggle  
- Type: Diabetes & Heart Disease datasets  

**Features include:**
- Demographics (Age, Gender)  
- Medical attributes (Glucose, Blood Pressure, Cholesterol)  
- Target variable (Disease presence)
